# RamXCat Portfolio 🚀

A high-end, terminal-inspired professional portfolio built with a **Java Spring Boot 3** backend and a premium, modern frontend experience.

![Hero Section Placeholder](src/main/resources/static/Portfolio%20icons%20photos/Image-1.png)

## ✨ Features

- **Java Spring Boot 3 Backend**: Robust serving of static assets and potential for dynamic project management.
- **Glassmorphism Design**: Sleek, modern UI with transparent, blurred backgrounds and subtle border glows.
- **Cinematic Animations**:
    - **Scroll Transitions**: Elements slide and fade in using professional `cubic-bezier` curves.
    - **Staggered Reveals**: Rhythmic cascading of skill cards and social links for a "wave" effect.
    - **Hero Choreography**: Synchronized logo ornaments with infinite floating and entrance animations.
- **Responsive Layout**: Designed to look stunning on all devices, from desktop to mobile.
- **Interactive UX**: Custom terminal-themed scrollbar and real-time scroll progress tracker.

## 🛠️ Tech Stack

**Backend:**
- Java 21
- Spring Boot 3.4.1
- Maven (Project Management)

**Frontend:**
- HTML5 (Semantic Structure)
- CSS3 (Vanilla + Custom Animations)
- JavaScript (Vanilla + Intersection Observer API)

## 🚀 Getting Started

To run this project locally, ensure you have **Java 21** and **Maven** installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RamXCat/Portfolio.git
   cd Portfolio
   ```

2. **Build the application:**
   ```bash
   mvn clean package
   ```

3. **Run the JAR:**
   ```bash
   java -jar target/portfolio-0.0.1-SNAPSHOT.jar
   ```

4. **Access the site:**
   Navigate to `http://localhost:8080` in your browser.

## 📁 Project Structure

- `src/main/java`: Backend controller and service logic.
- `src/main/resources/static/`: All frontend assets (HTML, CSS, JS, Images).
- `pom.xml`: Maven dependency and build configuration.
